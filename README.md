# Easy Point

**Easy Point** es una herramienta online que permite calcular las señales de PLC y generar automáticamente la memoria de control para proyectos HVAC (calefacción, ventilación y aire acondicionado). Pensada para ingenierías, técnicos e instaladores que desean agilizar la documentación técnica durante la fase de diseño o estudio.

> ⚠️ Esta herramienta está pensada como ayuda en fases preliminares de proyecto. Para el dimensionado y selección definitiva de equipos, es imprescindible contar con el asesoramiento de un profesional cualificado.

## ✨ Características

- Cálculo automático de señales de entrada/salida por bloque funcional.
- Generación de documentos en PDF (estudio de señales y memoria de control).
- Interfaz sencilla y orientada a la productividad.
- Compatible con navegadores modernos y ejecutable offline.
